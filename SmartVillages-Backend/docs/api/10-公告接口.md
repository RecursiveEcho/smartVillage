# 公告接口

> 本模块内每个接口已折叠显示，点击接口名展开详情。

## 本模块接口

- 村干部公告详情
- 编辑公告基础信息
- 删除公告
- 上架-下架公告
- 村干部审核公告
- 村干部分页查询公告
- 村干部新增公告
- 村干部待审核公告
- 审核历史列表
- 前台分页公告（仅已发布）
- 公告详细
- 热门公告

|&emsp;&emsp;auditUser|审核人|integer(int32)||
|&emsp;&emsp;viewCount|浏览次数|integer(int32)||
|&emsp;&emsp;createUser|创建人|integer(int32)||
