# 村务管理-村务事项-公示

> 本模块内每个接口已折叠显示，点击接口名展开详情。

## 本模块接口

- 根据id获取村务事项-公示详情

- 删除村务事项-公示
- 分页查询村务事项-公示列表
- 创建村务事项-公示
- 审核村务事项-公示

- 根据id获取村务事项-公示详情
