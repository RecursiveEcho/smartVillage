# 村务管理-民生服务

> 本模块内每个接口已折叠显示，点击接口名展开详情。

## 本模块接口

- 取消我的民生服务工单申请
- 管理端处理民生服务工单申请
- 管理端办结民生服务工单申请
- 管理端关闭工单
- 村民提交民生服务工单
- 获取民生服务工单列表
- 获取我的民生服务工单详情
- 管理端获取民生服务工单列表
- 管理端获取民生服务工单详情
- 后台申请统计
